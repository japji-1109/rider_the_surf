# rider_the_surf

🌊 WaveRiders – Surf Academy Website

A modern and responsive surf academy landing page built using HTML5 and CSS3.
WaveRiders is designed to promote surfing courses with a clean ocean-themed layout and interactive course cards.

📌 Project Overview

WaveRiders Surf Academy is a static front-end website that includes:

Navigation bar

Full-screen hero section with background overlay

About section

Course cards section

Bottom showcase image

Footer section

Responsive design

The website focuses on clean layout design, modern UI aesthetics, and responsive web development principles.

🚀 Features

✅ Full-screen hero with gradient overlay

✅ Ocean-themed color palette

✅ Wave-style section divider

✅ Course cards with hover animation

✅ Responsive layout using media queries

✅ Smooth and modern UI design

✅ Card-based layout structure

🛠️ Technologies Used

HTML5

CSS3

Flexbox

CSS Linear Gradient

CSS Transform & Transition

Media Queries

Box Shadow

🖥️ How to Run the Project

Download or clone the repository.

Open the project folder.

Double-click on index.html.

The website will open in your browser.

No installation required. This is a static front-end project.

📖 Website Sections Explanation
1️⃣ Navigation Bar

Contains:

Website logo (WaveRiders)

Navigation links (Home, Courses, About, Contact)

Styled with dark blue background

Hover underline effect on menu items

2️⃣ Hero Section

Full-screen background image

Gradient overlay using:

linear-gradient(rgba(), rgba())

Centered heading:

“Ride the Waves”

Subheading text

Call-to-action button (Join Now)

3️⃣ Wave Divider

Created using border-radius styling

Gives smooth wave-like transition between sections

4️⃣ About Section

Center-aligned heading and paragraph

Describes the academy’s mission

Simple clean layout

5️⃣ Our Courses Section

Contains 3 course cards:

Beginner Surfing

Intermediate Training

Pro Wave Mastery

Each card includes:

Image

Title

Hover lift animation using:

Large full-width background image
